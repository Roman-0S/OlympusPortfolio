LazyBug

LazyBug

LazyBug is a bug tracker that simplifies project and ticket organization. Tickets are managed by status, priority, and completion indicators, providing a clear view of progress. LazyBug also features a secure, role-based system to limit permissions. Users are assigned roles like Admin, Project Manager, Developer, and Submitter, each with specific permissions that allow each role to do their job efficiently.

The system was built with C# and .Net in 3 weeks. The front end was developed with HTML, Javascript and Bootstrap. The back-end utlizies Blazor and C#. The database was built with PostgreSQL and hosted on Railway. Please visit the site and try it out!

Stay Synced

Stay Synced

Stay Synced is an address book web application built with ASP.NET, C#, and SQL. Stay Synced streamlines communication by allowing you to easily email individual contacts or entire categories of contacts with just a few clicks. Organize your contacts efficiently and stay connected effortlessly, whether you're reaching out to a single person or a whole group.

The web app was built with C# and .Net in 1 week. The front end was developed with HTML, JavaScript, and Bootstrap. The back-end utilizes Blazor and C#. The database was built with PostgreSQL and hosted on Railway. Visit the site and try it out!
